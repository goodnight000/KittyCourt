# Sound Assets

## Required Files

### `deep-purr.mp3`
A seamless looping audio file of a deep cat purr for the meditation waiting screen.

**Recommended sources for free cat purr sounds:**
- [Pixabay](https://pixabay.com/sound-effects/search/cat%20purr/)
- [Freesound](https://freesound.org/search/?q=cat+purr)
- [Mixkit](https://mixkit.co/free-sound-effects/cat/)

Download a cat purr MP3 and save it as `deep-purr.mp3` in this folder.

**Ideal characteristics:**
- Duration: 10-30 seconds (it loops automatically)
- Low frequency, deep rumble
- Clean, no background noise
- Seamless loop points if possible
